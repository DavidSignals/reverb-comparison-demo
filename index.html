Interactive reverb analysis and listening comparison.

This web demo is designed to make the practical effect of the neural approach easier to evaluate. You can select a signal family, listen to the four rendered versions, inspect a real-time spectrum while any version is playing, and compare the four average spectral profiles directly in the browser.

The metrics section summarizes concrete values inspired by your previous study workflow, including correlation-style indicators and simple energy descriptors so the doctor gets both a listening demo and a structured comparison view.
